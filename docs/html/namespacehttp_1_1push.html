Detailed Description

Use HTTP gateways in push mode.

In this section, the processor chain operates in push mode. Events are produced upstream, and are pushed to the downstream processors.

You are advised to first look at PushLocal, which contains a single-machine simulation with detailed step-by-step documentation. You can then see the programs PushMachineA and PushMachineB, which perform the same computation, but across two independent programs that you can run on distinct computers.
