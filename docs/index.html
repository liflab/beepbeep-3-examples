Code Examples
A repository of 155 code examples for BeepBeep

This repository contains a large number of simple coding examples illustrating the use of the BeepBeep 3 event stream processor and the various palettes that come with it. All the code examples are heavily documented, with line-by-line explanations of what is being done.

What is BeepBeep?

BeepBeep is an event stream processing library. It has been developed by a team of researchers at Laboratoire d'informatique formelle, a research lab located at Université du Québec à Chicoutimi, Canada.

There exists extensive and detailed documentation about BeepBeep. This page only covers cod examples using BeepBeep. To learn more about the tool itself, please refer to these sources of information:

  • A complete, 300-page textbook that details the basic concepts and advanced features, hosted on GitBook and also available as an open-access publication.
  • BeepBeep's website for more references, such as download links and scientific publications
  • The API reference for BeepBeep's core and its standard palettes

How to use this repository

The source code repository is organized into Java packages. Each package is centered on a particular concept or functionality. For example, the basic package contains code examples using BeepBeep's core processors, while the plots package shows how to use processors and functions from the Plots palette.

Typically, an example is a file that contains a main() method that you can run. For each file, you can see the detailed source code directly from this website; this can be done by clicking on the link that is found at the bottom of each page. See for example the source code for one of the simplest examples.

The source code for all the examples can be downloaded from GitHub at: https://github.com/liflab/beepbeep-3-examples.

How to run the examples

The easiest way to run the examples is to open this whole repository into your favorite IDE, such as Eclipse or NetBeans. Simply locate a file with a main() method, read the associated comments in this file to understand what happens, and click Run.

All these examples require at least BeepBeep's core library, beepbeep-3.jar, in order to run. The latest release of this library can be downloaded from GitHub. Many sections of the repository focus on the use of one or more palettes and require additional JAR files. A big bundle of all these files can also be downloaded from GitHub.

If Ant is installed on your system, typing 

$ ant download-deps

from the root folder of this repository should take care of downloading and extracting all the required files and place them in the Source/dep folder.
