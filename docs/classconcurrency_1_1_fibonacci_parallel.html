Detailed Description

Computes the first N Fibonacci numbers.

This is done by pushing the integers 2, 3, … into a FibonacciProcessor that computes the 2nd, 3rd, … Fibonacci number.

This program is identical to FibonacciSequential except that the FibonacciProcessor is encased into a PushPipeline processor. This allows multiple calls to

push()

to be executed in parallel on multiple copies of the FibonacciProcessor.

On a machine with multiple cores, this program should run faster than FibonacciSequential for the same value of UPPER_BOUND. Note that the performance gain may become apparent only for a large value of this parameter. If only a small number of events are pushed the overhead of using multiple threads offsets the potential performance gains offered by concurrency.
