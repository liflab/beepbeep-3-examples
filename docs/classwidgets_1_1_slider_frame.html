Detailed Description

JFrame with a numerical slider that can be queried for its value.

This window is used in a couple of examples in this section, such as AverageSlider. The SliderFrame is implemented as a Source processor that has a JFrame as a member field. When asked for a new event (through method compute()), this processor returns a single integer corresponding to the current value of the slider.
