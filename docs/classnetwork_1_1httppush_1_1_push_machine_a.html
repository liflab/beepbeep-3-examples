Detailed Description

This is the same example as PushLocal, but with the "Machine A" and "Machine B" parts of the chain split into two independent programs, with some user interaction added.

You are advised to first go through the PushLocal example. This file corresponds to Machine A.

You can actually run this example on two physical machines if you wish. Simply start PushMachineA and PushMachineB on two computers, and use the IP address of Machine B in the URL you give to Machine A.
