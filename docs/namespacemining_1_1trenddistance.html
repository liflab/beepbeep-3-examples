Detailed Description

Evaluate an event stream based on a distance to a reference trend.

It is possible, however, to encapsulate this process into a GroupProcessor, which becomes a generic pattern whose actual computation is based on six parameters.

Depending on how these six parameters are instantiated, the TrendDistance processor computes different things.
